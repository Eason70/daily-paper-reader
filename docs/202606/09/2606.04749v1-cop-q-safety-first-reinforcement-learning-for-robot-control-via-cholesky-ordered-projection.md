## 摘要
安全机器人控制要求在满足安全约束的同时最大化回报。在离策略安全强化学习中，奖励和安全Q值通常由独立的评论家集成学习，每个目标独立处理不确定性。这种按目标处理的方式忽略了目标间的相关性，可能导致过于保守的价值估计，从而降低样本效率。为解决此问题，我们提出了乔列斯基有序投影Q学习（COP-Q），一种将目标间协方差纳入向量值Q值估计的安全优先方法。COP-Q在联合Q值空间中构建广义置信界，并利用乔列斯基分解以顺序形式编码目标优先级。这在不牺牲安全性的同时，自适应地减少对奖励目标的过度保守性。所得估计用于时间差分目标计算和智能体优化。COP-Q引入最小的计算开销，并易于与大多数现有深度Q学习框架兼容。在Brax机器人 locomotion 和Safety-Gymnasium安全导航上的实验（涵盖硬安全和软安全设置）表明，相对于代表性基线，COP-Q在实现强大安全性能的同时，具有竞争性或改进的样本效率。

## Abstract
Safe robot control requires maximizing return while satisfying safety constraints. In off-policy safe reinforcement learning, reward and safety Q-values are commonly learned by separate critic ensembles, with uncertainty handled independently for each objective. This objective-wise treatment neglects inter-objective correlation and can lead to overly conservative value estimates, thereby reducing sample efficiency. To address this issue, we propose Cholesky-Ordered Projection Q-learning (COP-Q), a safety-first method that incorporates inter-objective covariance into vector-valued Q-value estimation. COP-Q constructs a generalized confidence bound in the joint Q-value space and uses Cholesky factorization to encode objective priority in a sequential form. This preserves conservatism on safety while adaptively reducing excessive conservatism on the reward objective. The resulting estimate is used in both temporal-difference target computation and actor optimization. COP-Q incurs minimal computational overhead and is readily compatible with most existing deep Q-learning frameworks. Experiments on robot locomotion in Brax and safe navigation in Safety-Gymnasium, covering both hard- and soft-safety settings, demonstrate that COP-Q achieves strong safety performance together with competitive or improved sample efficiency relative to representative baselines.

### 1. 核心问题与整体含义（研究动机和背景）

- **研究背景**：安全强化学习（Safe RL）在机器人控制中需同时最大化回报（reward）和满足安全约束（safety）。离策略深度Q学习因样本效率高而被广泛采用，但其核心挑战是Q值估计中的过估计偏差。现有方法常使用多个独立Q网络（集成）来估计不确定性，并构造保守估计。
- **核心问题**：在安全RL中，回报和安全两个目标的Q值估计不是独立的，它们之间存在相关性。传统方法（如独立处理每个目标，或先标量化再处理不确定性）忽略了目标间协方差，导致：
  - 独立建模：结合两个目标的边际下界，当回报与安全负相关时，产生过度保守的估计，降低样本效率。
  - 标量化处理：对标量目标的保守性不能保证对安全目标的保守性，在负相关时可能高估安全，导致不安全行为。
- **论文含义**：提出一种能够利用目标间协方差、同时体现安全优先级的保守值估计方法，以平衡安全与样本效率。

### 2. 论文提出的方法论

- **核心思想**：COP-Q（Cholesky-Ordered Projection Q-learning）将回报和安全的Q值视为向量值函数，在联合Q值空间中构造广义置信界，并通过Cholesky分解将安全优先的顺序编码到投影过程中，从而在保持对安全的保守性的同时，利用协方差自适应地修正对回报目标的过度保守。
- **关键技术细节**：
  1. **广义多目标置信界**：将单目标Q值估计的高斯假设推广到多目标向量，认为评论家集成的Q向量服从多元高斯分布 \(N(\mu_Q, \Sigma_Q)\)。在置信椭圆上选取一个点作为保守估计，一般形式为 \(Q' = \mu_Q - \sqrt{\chi^2_N(p)} \times B u\)，其中 \(B\) 是 \(\Sigma_Q\) 的矩阵平方根因子，\(u\) 是标量化向量（单位范数）。
  2. **Cholesky有序投影**：为体现安全优先，利用Cholesky分解 \(\Sigma_Q = L L^\top\)（\(L\) 是下三角矩阵）。投影变为 \(Q' = \mu_Q - \sqrt{\chi^2_N(p)} \times L u\)。解释：安全目标仅受自身边际不确定性影响（因为\(L\)第一列对应安全，下三角结构使安全的第一项只乘自己），而回报目标同时受自身不确定性和与安全的协方差影响。这保证了安全保守性，同时利用协方差修正回报的过度保守。
  3. **算法流程**：在SAC（Soft Actor-Critic）框架中实现：
     - 更新评论家网络时，从目标评论家集成估计下一状态-动作对的均值 \(\hat{\mu}_Q\) 和协方差 \(\hat{\Sigma}_Q\)（使用有偏协方差估计，至少需要 \(N+1\) 个评论家），计算Cholesky因子 \(\hat{L}\)，构造TD目标：\(y_t = z_t + \gamma[\hat{\mu}_Q - \beta \hat{L} u - \alpha H]\)，其中 \(\beta = \sqrt{\chi^2_N(p)}\)，\(H\) 是最大熵项。
     - 更新演员时，最大化目标函数：\(J_\pi^{\text{total}} = E[ u^\top (\hat{\mu}_Q - \beta \hat{L} u) - \alpha H]\)。
  4. **计算开销**：仅需最少 \(N+1\) 个评论家（本文使用3个？但原文说“至少N+1”，N=2时恢复双Q），Cholesky分解计算效率高。超参数仅为 \(\beta\)（Mahalanobis距离），避免了为两个相关目标调两个超参数。
  5. **退化为已知方法**：当 \(N=1\) 时退化为标准单目标保守目标构建；当 \(N=2\) 时恢复经典的 clipped double-Q 学习。

- **数值稳定性**：Cholesky分解对协方差矩阵的条件数敏感。当目标相关性接近±1或边际标准差比值极端时，协方差矩阵可能病态，导致Cholesky因子对估计误差敏感，影响学习稳定性。

### 3. 实验设计

- **任务与场景**：

  - **T.3 安全导航（Safe Navigation）**：Safety-Gymnasium 中的 Goal2 和 Button2（最高难度），使用 Point 和 Car 机器人。需避开静态和动态障碍物，最大回合400步，成本限值10。场景设置与CVPO相同。
- **基准对比方法**：

### 6. 论文的主要结论与发现

1. COP-Q在硬安全任务中实现了与基线相当或更少的安全事故（尤其在人形机器人上大幅减少跌落），同时样本效率更高（回报曲线上升更快）。
2. 在软安全任务（Safe Velocity和Safe Navigation）中，COP-Q在满足成本约束的前提下实现了更高的回报，优于独立保守基线（如SACLag-UCB、CAL），因为它们过于保守；也优于单批评家基线（如SACLag，其违背约束）。
3. 相关性分析表明：当回报与安全负相关越强（如人形机器人），COP-Q的优势越明显。当相关性接近零，优势较小。
4. 消融实验显示：当安全信号为二值稀疏信号时，协方差矩阵容易病态，COP-Q优势消失甚至不稳定，揭示了局限。

- **实验设计全面**：覆盖不同安全类型、机器人形态、任务复杂度，对比多种基线，并辅以相关性分析和消融，论证充分。

### 8. 不足与局限

- **局限性1**：对协方差矩阵估计的数值稳定性敏感。当安全信号稀疏、二元、近乎同质时（如纯生存奖励），协方差矩阵可能病态，导致Cholesky因子不稳定，削弱算法优势甚至失效（消融实验证实）。
- **局限性2**：当前方法仅捕捉总估计不确定性，未区分认知不确定性和偶然不确定性。未来可考虑更细致的分解。
- **局限性3**：实验仅在两个仿真环境（Brax、Safety-Gymnasium）进行，未在真实机器人或更复杂场景（如多机器人、连续动作空间）验证。泛化性有待进一步证明。
- **局限性4**：对于不同任务需手动调整 \(\beta\)（locomotion用1，navigation用0.5），缺乏自适应的调参机制。
- **局限性5**：未与其他多目标RL方法（如CAPQL等）在安全场景下直接对比，仅对比了传统CRL方法。

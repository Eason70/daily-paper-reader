## 摘要
在复杂的连续控制强化学习任务中，多模态最优动作往往与不确定的多模态回报分布相伴，这使得可靠的值估计和多模态探索变得具有挑战性。现有使用单峰高斯的值估计方法限制了表达能力并产生有偏估计。近年来，生成式策略可以表示多模态动作，但往往坍缩到少数模态，且对动作空间中高值区域的探索不足。受这些挑战的启发，我们提出了Dual-Flow RL，一种统一的演员-评论家框架，通过条件流匹配（CFM）联合建模连续回报分布和多模态策略分布。该设计支持可靠的值估计和持续的多模态探索。为进一步增强探索，我们引入了熵-协方差探索调节器（ECER），利用策略熵和动作不确定性协方差实现状态感知的探索调节。在DeepMind Control Suite和Humanoid-Bench上的实验表明，Dual-Flow RL在大多数任务上达到了最先进的性能，显著优于先前的基于扩散和基于流的方法。

## Abstract
In complex continuous-control reinforcement learning tasks, multimodal optimal actions often coincide with uncertain, multimodal return distributions, making reliable value estimation and multimodal exploration challenging. Existing value estimation methods using unimodal Gaussians restrict expressiveness and yield biased estimates. Recent generative policies can represent multimodal actions but often collapse to a few modes and under-explore high-value areas of the action space. Motivated by these challenges, we propose Dual-Flow RL, a unified actor-critic framework that jointly models a continuous return distribution and a multimodal policy distribution using conditional flow matching (CFM). This design supports reliable value estimation and sustained multimodal exploration. To further enhance exploration, we introduce an Entropy-Covariance Exploration Regulator (ECER) that enables state-aware exploration regulation leveraging policy entropy and action-uncertainty covariance. Experiments on DeepMind Control Suite and Humanoid-Bench show that Dual-Flow RL achieves state-of-the-art performance on most tasks, significantly outperforming prior diffusion-based and flow-based methods.

## 论文核心问题与整体含义（研究动机和背景）

- **核心问题**：在复杂连续控制任务中，最优动作往往呈现多模态分布，且对应的回报分布也是多模态且高不确定性的。现有的价值估计方法通常采用单峰高斯分布，限制了表达能力，导致有偏估计；而最近提出的生成式策略（如扩散或流模型）能够表示多模态动作，但容易坍缩到少数高值模式，对动作空间中其他高价值区域的探索不足。
- **背景**：传统强化学习方法如SAC、TD3使用确定性或对角高斯策略，无法捕捉多模态行为；分布性RL方法如C51、IQN虽能建模回报分布，但受限于固定支撑集或量化误差；生成式策略虽表达力强，但缺乏状态自适应的探索机制。因此需要一个统一框架同时解决价值估计偏差和探索不足问题。

## 方法论

- **核心思想**：构建一个统一的Actor-Critic框架，使用条件流匹配（Conditional Flow Matching, CFM）同时建模连续回报分布（Critic）和多模态策略分布（Policy），并引入状态感知的探索调节器（Entropy-Covariance Exploration Regulator, ECER）。
- **关键技术细节**：
  1. **Flow-based分布性Critic**：将回报分布建模为状态-动作条件分布 \(Z(s,a)\)，通过ODE从基分布 \(N(0,1)\) 沿时间 \(t\in[0,1]\) 演化得到终端样本 \(z_1\sim Z(s,a)\)。训练时最小化CFM损失，使ODE生成的分布与TD目标分布对齐。
  2. **Flow-based策略**：类似地，将策略建模为状态条件流，从 \(a_0\sim N(0,I)\) 演化得到动作 \(a_\pi\)。训练目标包括最大化期望回报和通过优势加权引导策略向高回报行为对齐。
  3. **熵-协方差探索调节器（ECER）**：
     - 策略熵 \(H(s)\)：通过对GMM近似策略分布进行熵估计。
     - 动作-不确定性协方差 \(\rho(s)\)：计算对数密度与回报不确定性之间的相关系数。
     - 构建门控机制：熵门 \(g_H\) 和协方差门 \(g_D\)，联合得到全局门 \(g = g_H g_D\)，用于动态调整正则化系数 \(\lambda_{\text{eff}}\)。
     - 最终ECER损失：最大化优势加权的对数似然，同时对探索尺度 \(\sigma(s)\) 施加L2正则化。
- **算法流程**：交替执行采样（使用探索增强动作 \(a_e = a_\pi + \sigma(s)\odot\epsilon\)）和更新（同时更新Critic、Policy、ECER及门控值）。
